# Changelog

## [0.0.2] - 2026-03-23

### Fixed
- Pinned tabs are no longer moved into tab groups

## [0.0.1] - 2025-12-03

### Added
- Initial release
- Automatically group tabs by domain using configurable rules
- Popup with enable/disable toggle and group overview
- Options page to add, edit, and delete groups (title, color, domains)
- Import/export configuration as JSON
- Badge on extension icon showing number of active groups
- Config stored in `chrome.storage.sync`
